## Telegram Affiliate Menu Bot

Python-based Telegram bot that provides a simple interactive menu for promoting products
and mobile applications using affiliate links.

### Features
- Keyword-based commands
- Inline keyboard menu
- Affiliate link distribution
- Async Telegram bot (aiogram)

### Use Cases
- Affiliate marketing campaigns
- Product and app promotion
- Telegram channel automation
- Lightweight marketing bots

### Tech Stack
- Python 3
- aiogram
- AsyncIO

### Security Notice
Bot token must be stored in environment variables.
Never commit API tokens or sensitive credentials.

This project is for demonstration and educational purposes.
Users are responsible for complying with affiliate program terms.
